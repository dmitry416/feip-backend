## Как запустить?
```bash
docker compose build
docker compose up -d
```
После следует подождать, пока все запустится, перед тестированием.

## Тестирование
В папке tests есть тест test-import.ps1, запусив который будет наглядно отправлен запрос и после отобразится содержимое таблиц.
